# Django Sample Website

A modern, responsive, and SEO-friendly **Django website template** built with **Python**, **Django**, and **Bootstrap**. This project provides a clean foundation for building business websites, blogs, affiliate marketing platforms, and content-driven web applications.

It demonstrates reusable Django templates, responsive layouts, static and media file handling, and a scalable project structure that can be extended for production-ready applications.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Subashmca2017/Sample_Website.git
```

### Navigate to the Project

```bash
cd Sample_Website/simple_website
```

### Create a Virtual Environment

```bash
python -m venv venv
```

### Activate the Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Linux / macOS

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Database Migrations

```bash
python manage.py migrate
```

### Start the Development Server

```bash
python manage.py runserver
```

Visit:

```
http://127.0.0.1:8000/
```
